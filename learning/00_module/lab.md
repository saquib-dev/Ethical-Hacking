# Module 00: Practical Exercise

### Task 1: Legal Research
1. Search for the primary cybersecurity law in your country (e.g., CFAA in USA, IT Act 2000 in India).
2. Identify one specific action that is considered a crime under that law.
3. Write down why "intent" doesn't always protect you from legal action.

### Task 2: Creating a Rules of Engagement (ROE)
Imagine you are hired to test a small business website. Draft a simple ROE document including:
- **Target**: (e.g., `www.example-business.com`)
- **Forbidden Actions**: (e.g., "Do not crash the server," "Do not delete database records")
- **Contact Person**: Who do you call if the server goes down?
- **Timeline**: When does the test start and end?

### Task 3: Environment Verification
1. Open your Kali terminal.
2. Run `whoami` and `ip addr`.
3. Save the output to a file called `env_check.txt` using the `>` operator.
