# Module 00: Conceptual Guide

While this module is theoretical, it is important to familiarize yourself with the environment you will be using.

### Basic System Checks in Kali
Run these in your terminal to ensure your environment is ready:
- `whoami`: Check which user you are currently logged in as.
- `uname -a`: Check the kernel version and system architecture.
- `lsb_release -a`: Verify the version of Kali Linux installed.
- `ip addr`: Confirm your network interface is active.
