# Module 1: Container Basics (Docker)

## What is a Container?
A container is a lightweight version of a VM. Instead of virtualizing the hardware, it virtualizes the Operating System.

## Key Concepts:
- **Image**: The blueprint for a container.
- **Container**: The running instance of an image.
- **Docker Daemon**: The process that manages containers on the host.

## Why it's a Target:
Many companies put their apps in Docker. If the container is misconfigured, a hacker can "escape" the container and take over the entire Host OS.
