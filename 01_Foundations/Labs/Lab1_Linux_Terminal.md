# Lab 1: Linux Terminal Practice

## Goal: Get comfortable with the command line.

## Commands to Try:
1. `pwd` : (Print Working Directory) - Where am I?
2. `ls` : (List) - What files are here?
3. `cd <folder>` : (Change Directory) - Go inside a folder.
4. `mkdir my_test` : Create a new folder.
5. `touch hello.txt` : Create a blank file.
6. `echo "Hello Hacker" > hello.txt` : Put text into a file.
7. `cat hello.txt` : Read the file.
8. `rm hello.txt` : Delete the file.
9. `sudo apt update` : Update the software list (requires password).

## Success Check:
- [ ] I can navigate folders.
- [ ] I can create and delete files.
- [ ] I know how to use `sudo`.
