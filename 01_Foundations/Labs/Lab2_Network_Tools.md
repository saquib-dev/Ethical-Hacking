# Lab 2: Network Tools Practice

## Goal: See how your machine connects to the world.

## Tools to Try:
1. `ifconfig` (or `ip a`): Find your own IP address.
2. `ping google.com`: Check if you can reach a server.
3. `whois google.com`: Find out who owns a domain.
4. `dig google.com`: See the DNS records (IP of the website).
5. `netstat -tunlp`: See which ports are open on your own machine.

## Success Check:
- [ ] I found my private IP.
- [ ] I successfully pinged a website.
- [ ] I can see my open ports.
