# Module 1: Linux Basics

## Why Linux?
Almost all hacking tools are built for Linux. Most servers in the world run Linux.

## The Terminal (CLI)
Instead of clicking icons, you type commands. It is faster and more powerful.

## Key Concepts:
- **Root**: The "God Mode" user. Has power to do everything.
- **Sudo**: "SuperUser Do". Used to run a command as root.
- **Filesystem**: Everything in Linux is a file.
- **Permissions**: Who can read, write, or execute a file.
