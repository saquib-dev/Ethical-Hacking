# Module 2: Networking Basics

## How Computers Talk
Computers use **IP Addresses** to find each other.
- **Public IP**: Your address on the global internet.
- **Private IP**: Your address inside your home WiFi.

## Ports (The "Doors")
A computer has 65,535 ports. Think of them as doors to different services:
- **Port 80**: HTTP (Websites)
- **Port 443**: HTTPS (Secure Websites)
- **Port 22**: SSH (Remote Access)
- **Port 21**: FTP (File Transfer)

## Protocols
The "language" computers speak.
- **TCP**: Reliable (checks if data arrived).
- **UDP**: Fast (doesn't check if data arrived - used for gaming/streaming).
