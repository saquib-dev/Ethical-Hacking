# Module 2: Python for Penetration Testing

## Key Libraries every Hacker knows:
1. **Requests**: For interacting with Web APIs and websites.
2. **Scapy**: For crafting custom network packets (The "Swiss Army Knife" of networking).
3. **BeautifulSoup**: For parsing HTML and extracting data (Web Scraping).
4. **Paramiko**: For automating SSH connections.
5. **PwnTools**: The industry standard for CTFs and Binary Exploitation.

## The Workflow:
`Find Bug` $\rightarrow$ `Write Proof-of-Concept (PoC) in Python` $\rightarrow$ `Automate the Attack`.
