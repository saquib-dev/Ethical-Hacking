# Burp Suite Mastery Plan (4 Weeks)

Burp Suite is the most important tool for web hacking. This plan moves you from "Proxy" to "Professional".

## Week 1: The Basics (Interceptor & Repeater)
- **Goal**: Understand the HTTP request/response cycle.
- **Tasks**:
  - Master "Intercept" to modify requests on the fly.
  - Use "Repeater" to test different payloads without re-sending from the browser.
  - Learn to use the Burp Browser.

## Week 2: Automation (Intruder)
- **Goal**: Automate attack patterns.
- **Tasks**:
  - Master the 4 attack types: Sniper, Battering Ram, Pitchfork, Cluster Bomb.
  - Create custom wordlists for directory brute-forcing.
  - Use "Grep-Match" to find specific responses in thousands of requests.

## Week 3: Out-of-Band Attacks (Collaborator)
- **Goal**: Find "Blind" vulnerabilities.
- **Tasks**:
  - Set up Burp Collaborator.
  - Use it to detect Blind SSRF and Blind XXE.
  - Understand how DNS/HTTP interactions prove a vulnerability exists.

## Week 4: Extensibility (BApp Store)
- **Goal**: Expand Burp's power.
- **Tasks**:
  - Install and use **Logger++** and **Autorize** (for IDOR/BOLA testing).
  - Learn how to use **Turbo Intruder** for high-speed race conditions.
