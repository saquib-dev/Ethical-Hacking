# Mini-Boss 1: The Silent Gateway

## Goal: Gain access to a server using ONLY a custom Python script.

## The Challenge:
You are given a target IP. The server has a custom port (e.g., 8888) that requires a specific 4-digit PIN to allow a shell.
- You cannot use Metasploit.
- You cannot use Nmap for the exploit.
- You must write a Python script to brute-force the 4-digit PIN and automatically trigger the shell once found.

## Success Criteria:
- [ ] Script successfully finds the PIN.
- [ ] Script automatically captures the shell.
- [ ] No manual intervention during the attack.
