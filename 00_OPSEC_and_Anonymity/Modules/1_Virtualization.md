# Module 1: Virtualization (Staying Safe)

## What is a VM?
A Virtual Machine (VM) is like a "computer inside your computer." 
If you make a mistake or get a virus inside the VM, your main computer (the Host) stays safe.

## Why use it for Hacking?
1. **Isolation**: Keeps your main system clean.
2. **Snapshots**: You can save the state of your VM. If you break something, just "go back in time" with one click.
3. **Multiple OS**: You can run Kali Linux (hacking OS) and Windows (target OS) at the same time.

## Recommended Software
- **VirtualBox** (Free and Easy)
- **VMware Workstation Player** (Fast and Stable)
