# Lab 1: Setting Up Your Safe Zone

## Goal: Install a VM and Kali Linux.

## Steps:
1. **Download VirtualBox**: Go to virtualbox.org and install it.
2. **Download Kali Linux**: Go to kali.org -> Get Kali -> Virtual Machines.
3. **Import Kali**: Open VirtualBox -> File -> Import Appliance -> Select the Kali file.
4. **Test it**: Start the VM. 
   - Default Username: `kali`
   - Default Password: `kali`
5. **Take a Snapshot**: Right-click the VM -> Snapshot -> Take. Name it "Fresh Install".

## Success Check:
- [ ] Kali Linux is running.
- [ ] You can open the terminal.
- [ ] You have a "Fresh Install" snapshot.
