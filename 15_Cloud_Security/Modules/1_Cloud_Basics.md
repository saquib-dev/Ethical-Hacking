# Module 1: Cloud Architecture Basics

## What is the Cloud?
The cloud is just someone else's computer. However, instead of managing hardware, you manage "Services" via an API.

## Key Concepts:
1. **IAM (Identity and Access Management)**: The most important part of cloud security. It defines WHO can do WHAT.
2. **S3 Buckets (AWS) / Blob Storage (Azure)**: Places where companies store files. If misconfigured, they are public to the whole internet.
3. **Compute Instances**: Virtual machines (EC2 in AWS). They often have "Roles" attached to them that give them permissions.
