# Module 1: Introduction to Active Directory (AD)

## What is Active Directory?
Active Directory is a database and set of services that allow a company to manage users, computers, and permissions in one central place.
- **Domain Controller (DC)**: The server that manages the AD database. It's the "Heart" of the network.
- **Domain**: The group of computers and users managed by the DC.
- **Forest**: A group of domains.

## Why is AD the main target?
Because if you compromise the Domain Controller, you have "God Mode" over every single computer and user in the entire company.
