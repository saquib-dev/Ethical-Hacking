# Final Engagement Report Template

## 1. Executive Summary
- **Objective**: What were you trying to achieve?
- **Overall Risk**: (Critical/High/Medium/Low)
- **Outcome**: Did you find the Obsidian Key?

## 2. Attack Chain (The Story)
Describe the flow of the attack.
*Step 1: Recon $\rightarrow$ Step 2: Initial Access $\rightarrow$ Step 3: Pivot $\rightarrow$ Step 4: Escalation $\rightarrow$ Step 5: Exfiltration.*

## 3. Technical Findings
For every vulnerability found:
- **Vulnerability Name**: (e.g., Unrestricted File Upload)
- **Location**: (URL or IP)
- **Impact**: (e.g., Remote Code Execution)
- **Proof of Concept (PoC)**: (Commands used, screenshots)
- **Remediation**: How should the company fix this?

## 4. Conclusion
Final thoughts on the security posture of SecureCorp.
