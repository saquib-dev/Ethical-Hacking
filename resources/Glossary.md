# Cybersecurity Glossary

A centralized repository of key terms and definitions used throughout this curriculum.

## A-Z Definitions

### A
- **Active Directory (AD):** A directory service developed by Microsoft for Windows domain networks.

### B
- **Brute Force:** An attack that attempts every possible combination of passwords or keys until the correct one is found.

### C
- **C2 (Command and Control):** The infrastructure used by an attacker to send commands to compromised systems.

### D
- **DoS (Denial of Service):** An attack intended to shut down a machine or network, making it inaccessible to its intended users.

### E
- **EDR (Endpoint Detection and Response):** Tools that monitor end-user devices to detect and respond to cyber threats.

### L
- **LFI (Local File Inclusion):** A vulnerability where an application includes a file from the local server, potentially allowing an attacker to read sensitive files.

### P
- **Privilege Escalation:** The act of gaining higher-level permissions on a system (e.g., moving from a standard user to root/administrator).

### R
- **ROP (Return-Oriented Programming):** An advanced exploitation technique used to execute code in the presence of security defenses like NX/DEP.

### S
- **SQL Injection (SQLi):** A web security vulnerability that allows an attacker to interfere with the queries that an application makes to its database.

### X
- **XSS (Cross-Site Scripting):** A vulnerability that allows an attacker to inject malicious scripts into web pages viewed by other users.
