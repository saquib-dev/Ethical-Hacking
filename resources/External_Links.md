# External Learning Resources

Curated links to platforms and documentation for further study and practice.

## 🛠️ Practice Labs
- [Hack The Box](https://www.hackthebox.com/) - Advanced labs and machines.
- [TryHackMe](https://tryhackme.com/) - Guided learning paths for beginners.
- [VulnHub](https://www.vulnhub.com/) - Downloadable vulnerable VMs.

## 📚 Documentation & Standards
- [OWASP Top 10](https://owasp.org/www-project-top-ten/) - The definitive guide to web application security.
- [MITRE ATT&CK](https://attack.mitre.org/) - A globally-accessible knowledge base of adversary tactics and techniques.
- [CVE (Common Vulnerabilities and Exposures)](https://cve.mitre.org/) - List of publicly disclosed cybersecurity vulnerabilities.

## 📖 Learning Portals
- [PortSwigger Web Security Academy](https://portswigger.net/web-security) - Free high-quality web hacking training.
- [Kali Linux Documentation](https://www.kali.org/docs/) - Official docs for the industry-standard OS.
